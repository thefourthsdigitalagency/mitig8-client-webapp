# Mitig8 Client WebApp – Orchestrated Repo (Mitig8_11 v6)

This folder is intended to be the **NEW Mitig8 repo root**, for example:

- `/Users/dominicnel/mitig8_client_webapp`

Inside it you will find:

- `mitig8_11/` – Orchestration bundle for TRAE (milestones, prompts, docs).
- (Later) `client/`, `api/`, `infrastructure/terraform/` – The actual application code and Terraform.

Open **this folder** in TRAE, not the `mitig8_11` folder directly.

Then, in TRAE, use the `mitig8_11/FIRST_RUN_PROMPT.txt` content as your initial instruction.
